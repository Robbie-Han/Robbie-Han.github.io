# VSCode 快捷键
### 打开控制台： cmd + shift + y
### 文件名搜索： cmd + p
### 打开新的窗口: cmd + shift + n
### 关闭当前文件：cmd + W
<!--more-->

### 移动当前行：opt + 上/下
### 复制当前行：opt + shift + 上/下
### 在当前行下添加一行：cmd + enter
### 跳到匹配的括号处：cmd + shift + \
### 跳到页尾：cmd + 下
### 选中当前行： cmd + i
### 注释块： opt + shit + a
### 替换：cmd + opt + f
### 移除光标：cmd + U
